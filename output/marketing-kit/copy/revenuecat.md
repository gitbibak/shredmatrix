# RevenueCat / No Paywall Statement

Full Balance için RevenueCat tarafında ücretli ürün, abonelik, entitlement, offering veya paywall hazırlanmayacak.

Bu dosyanın amacı monetizasyon planı yazmak değil; tam tersine ürün sözünü netleştirmek:
`Full Balance tamamen ücretsizdir. Her zaman ücretsiz kalacaktır. Asla ücretli olmayacak.`

## Approved Message

Headline:
`Free forever. No paywall.`

Subheadline:
`Full Balance is built around a simple promise: every core wellness flow stays free.`

Feature list:

- `No subscription` — No monthly plan, no yearly plan, no trial countdown.
- `No premium wall` — Nutrition, workouts, progress, achievements and profile remain open.
- `No card required` — Start from the web app without payment friction.

## If A RevenueCat Field Requires Copy

Use RevenueCat only as a place to document that there is no paid offering.

Do:

- `Free forever`
- `No subscription`
- `No card required`
- `No premium wall`
- `$0 forever`

Do not:

- `Upgrade`
- `Plus`
- `Pro`
- `Premium`
- `Trial`
- `Subscribe`
- `Monthly`
- `Yearly`
- `Unlock`
- `Limited free plan`

## Internal Note

RevenueCat Paywalls are designed for configurable purchase screens tied to offerings. Because Full Balance is explicitly free forever, do not configure products, packages, entitlements, experiments, intro offers or placement rules.

If a future teammate asks where the monetization plan is, point them here: the monetization plan is not to monetize access.
